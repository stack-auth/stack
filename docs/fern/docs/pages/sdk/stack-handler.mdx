# StackHandler

Renders the appropriate authentication or account-related component based on the current route.

## Props

- `app`: `StackServerApp<HasTokenStore>` - The Stack server application instance.
- `params`: `{ stack?: string[] }` - The route parameters, where `stack` is an array of path segments.
- `searchParams`: `Record<string, string>` - The URL search parameters.
- `fullPage`: `boolean` - Whether to render the component in full-page mode.
- `componentProps`: `{ [K in keyof Components]?: Parameters<Components[K]> }` - Props to pass to the rendered components.

## Example

```tsx
import { StackHandler } from '@stackframe/stack';

export default function Page({ params, searchParams }) {
  return (
    <StackHandler
      app={myStackServerApp}
      params={params}
      searchParams={searchParams}
      fullPage={true}
      componentProps={{
        SignIn: { /* SignIn component props */ },
        SignUp: { /* SignUp component props */ },
        // ... other component props
      }}
    />
  );
}
```
