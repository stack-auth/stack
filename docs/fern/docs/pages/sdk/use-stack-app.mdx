# useStackApp

A custom hook for managing Stack App state in your application.

## Props

- `selectedStackApp`: `StackApp | undefined` - The currently selected Stack App.
- `stackApps`: `StackApp[]` - An array of available Stack Apps.
- `setSelectedStackApp`: `(stackApp: StackApp) => void` - Function to set the selected Stack App.
- `setStackApps`: `(stackApps: StackApp[]) => void` - Function to set the array of Stack Apps.

## Example

```tsx
import { useStackApp } from '@stackframe/stack';

function StackAppManager() {
  const {
    selectedStackApp,
    stackApps,
    setSelectedStackApp,
    setStackApps
  } = useStackApp();

  // Example usage
  useEffect(() => {
    // Fetch stack apps and set them
    const fetchedApps = fetchStackApps();
    setStackApps(fetchedApps);
  }, []);

  const handleSelectApp = (app) => {
    setSelectedStackApp(app);
  };

}
```
