# SignUp

A component that renders a sign-up page with various customization options.

## Props

- `fullPage` (optional): `boolean` - If true, renders the sign-up page in full-page mode.
- `automaticRedirect` (optional): `boolean` - If true, automatically redirects the user after successful sign-up.
- `noPasswordRepeat` (optional): `boolean` - If true, removes the password confirmation field.
- `extraInfo` (optional): `React.ReactNode` - Additional information to display on the sign-up page.
- `firstTab` (optional): `'magic-link' | 'password'` - Determines which tab is initially active.

## Example

```tsx
import { SignUp } from '@stackframe/stack';

// Basic usage
<SignUp />

// Full page sign-up with automatic redirect and magic link as the first tab
<SignUp
  fullPage={true}
  automaticRedirect={true}
  firstTab='magic-link'
/>

// Custom sign-up with extra information
<SignUp
  noPasswordRepeat={true}
  extraInfo={<p>Join our community today!</p>}
/>
```
