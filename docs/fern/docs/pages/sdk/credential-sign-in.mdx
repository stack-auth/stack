---
title: <CredentialSignIn />
---

A component that renders a sign-in form with email and password fields.

## Props

This component does not accept any props.

## Example

```tsx
import { CredentialSignIn } from '@stackframe/stack';

function SignInPage() {
  return (
    <div>
      <h1>Sign In</h1>
      <CredentialSignIn />
    </div>
  );
}
```
